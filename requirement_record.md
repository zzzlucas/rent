# 需求记录 (Complex Requirements)

## 租房房源管理系统 (Rental Property Management System)

### 概述
为拥有数百个房源的房东设计的高效管理与维护系统。初期仅涉及前端设计，重点在于高美观度、易用性和高效的管理工作流。

### 核心功能需求
1. **仪表盘 (Dashboard)**: 
   - 房源概览（总数、出租率、空置数）。
   - 财务摘要（本月应收、实收、欠费）。
   - 待办事项（即将到期的合同、报修申请）。
2. **房源管理 (Property Management)**:
   - 列表/网格视图。
   - 详细信息（地址、租金、户型、配套设施）。
   - 状态追踪（空置、已租、维修、非业主房源）。
   - **高密度房态看板**: 2D 交互式视图，支持秀湖花苑大体量楼盘（如 21层/28室布局）。
   - **批量管理**: 支持按楼层或多选房间，批量标记为“托管中”或“非业主房源”，快速圈定管理范围。
3. **租客与合同管理 (Tenant & Contract)**:
   - 租客信息档案。
   - **AI 智能合同录入**: 支持拍摄/上传纸质合同图片，通过 AI 模型识别关键条款（租金、租期、租客姓名等）并自动填表。
   - 合同生命周期管理（签约、续约、到期提醒、退租）。
   - 电子附件存档（身份证、交房清单图片等）。
4. **报修维护 (Maintenance)**:
   - 报修单工单系统。
   - 维护记录追踪。
5. **财务流水 (Finances)**:
   - 租金缴纳状态。
   - 水电煤气费管理。
   - 支出记录（维修成本、中介费等）。
6. **数据中心与外部对接 (Data Hub & Integration)**:
   - **批量导入**: 支持将 Excel/CSV 格式的现有房源和租客数据一键导入。
   - **报表导出**: 提供各维度数据的 Excel/PDF 导出功能，便于财务对账。
   - **标准化模板**: 提供房源录入标准模板，方便跨平台迁移。
   - **对接预留**: 预留第三方支付与合同签署系统的对接入口。

### 设计风格
- 现代、高端、富有科技感。
- **全平台响应式设计**: 采用单套方案适配 PC、平板与手机，确保数据同步、体验一致。
- 动态交互效果（微动画）。
- 使用高对比度、清晰的排版。

### 技术栈
- Vue 3 + TypeScript
- Vite
- Vanilla CSS (with CSS Variables)
- Lucide Vue Next (Icons)
- Motion (for Vue)
